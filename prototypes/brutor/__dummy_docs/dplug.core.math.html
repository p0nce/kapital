  • logmap

    Declaration

    pure T logmap(T)(T t, T min, T max);

    Map the [0..1] range to [min..max] logarithmically.

    Parameters

    T t

    Interpolating value from [0 to 1]. UB if out of range.

    T min

    Value corresponding to t = 0. Must be > 0.

    T max

    Value corresponding to t = 1. Must be > 0.

    Note: You can totally have a max that is smaller than min. In this case, the range mapped with more accuracy will be small values (around max not min).

  • expDecayFactor

    Declaration

    pure nothrow @nogc double expDecayFactor(double timeConstantSecs, double samplerate);

    Gets a factor for making exponential decay curves, which are also the same thing as a 6dB/oct lowpass filter.

    Parameters

    double timeConstantSecs

    Time after which the amplitude is only 37% of the original.

    double samplerate

    Sampling rate.

    Return Value

    Multiplier for this RC time constant and sampling rate. Use it like this: smoothed += (v - smoothed) * result;

    Note: Using fast_exp yield a decay-factor within -180 dB (at 384000hz) of the one obtained with expm1. The alleged inaccuracies of plain exp just did not show up so we don't prefer expm1 anymore. This doesn't change the length of an iterated envelope like ExpSmoother. Actually, small variations of a decay factor often results in a misleadingly large RMS difference, that doesn't actually change the sound quality.

  • convertDecibelToLinearGain

    Declaration

    pure nothrow @nogc @safe float convertDecibelToLinearGain(float dB);
    pure nothrow @nogc @safe double convertDecibelToLinearGain(double dB);

    Converts from decibels (dB) to linear gain (aka. voltage).

    Parameters

    Return Value

    A voltage value, linear gain.

    Note: 0 dB is 1.0 in linear gain. -6 dB is 0.5 in linear gain. -inf dB is 0.0 in linear gain.

    Precision: This uses fast_exp which under normal conditions has a peak error under -135dB over the useful range. However, keep in mind that different exp function sound differently when modulated.
