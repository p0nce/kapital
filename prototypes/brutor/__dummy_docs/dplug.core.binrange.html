dplug.core.binrange

Binary parsing and emitting from input ranges, or to output ranges.

License

http://www.boost.org/LICENSE_1_0.txt

It is used internally by Dplug, and also by the user for emitting and parsing plugin state chunks.

  • skipBytes

    Declaration

    nothrow @nogc void skipBytes(ref const(ubyte)[] input, int numBytes, bool* err);

    Skip bytes while parsing an input slice.

    Parameters

    const(ubyte)[] input

    ubyte input range.

    int numBytes

    Number of bytes to skip.

    bool* err

    true if read error.

    Return Value

    On success, *err is set to false. On failure, *err is set to true, and the input range cannot be used anymore.

  • popBE

    Declaration

    T popBE(T)(ref const(ubyte)[] input, bool* err);
    T popLE(T)(ref const(ubyte)[] input, bool* err);

    Reads a base type from input slice. popLE parses bytes in little-endian order. popBE parses bytes in big-endian order.

    Parameters

    const(ubyte)[] input

    ubyte input range.

    bool* err

    true if read error.

    Supported types: byte, ubyte, short, ushort, int, uint, long, ulong, float, double

    Return Value

    On failure, *err is set to true, return 0. The input range cannot be used anymore.

    void writeLE(T, R)(ref R output, T n) if (isOutputRange!(R, ubyte));

    Writes a big-endian/little-endian base type to output range.

    Parameters

    R output

    ubyte output range.

    T n

    A base type to write.

    Supported types: byte, ubyte, short, ushort, int, uint, long, ulong, float, double

    Warning: Doesn't report write errors.
