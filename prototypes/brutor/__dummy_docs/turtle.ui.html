  • MicroUI

    Declaration

    class MicroUI;

    Provides immediate UI functionality in turtle. This is the original microUI but modified a bit:

    • replaced free function by object
    • gets passed a buffer and Font
    • clean-up of API

    Discussion

    You can get the MicroUI object with the Game.ui() call.
