  • Matrix

    Declaration

    struct Matrix(T, int R, int C);

    Generic non-resizeable matrix with R rows and C columns. Intended for 3D use (size 3x3 and 4x4).

    Important: Matrices here are in row-major order whereas OpenGL is column-major.

    Parameters

    T

    type of elements

    R

    number of rows

    C

    number of columns
