  • _mm_mpsadbw_epu8

    Declaration

    @trusted __m128i _mm_mpsadbw_epu8(int imm8)(__m128i a, __m128i b);

    Compute the sum of absolute differences (SADs) of quadruplets of unsigned 8-bit integers in a compared to those in b, and store the 16-bit results in dst. Eight SADs are performed using one quadruplet from b and eight quadruplets from a. One quadruplet is selected from b starting at on the offset specified in imm8[1:0]. Eight quadruplets are formed from sequential 8-bit integers selected from a starting at the offset specified in imm8[2].

    Declaration

    @trusted __m128d _mm_round_pd(int rounding)(__m128d a);

Rounding is done according to the rounding[3:0] parameter, which can be one of: (MM_FROUND_TO_NEAREST_INT |MM_FROUND_NO_EXC) // round to nearest, and suppress exceptions (MM_FROUND_TO_NEG_INF |MM_FROUND_NO_EXC) // round down, and suppress exceptions (MM_FROUND_TO_POS_INF |MM_FROUND_NO_EXC) // round up, and suppress exceptions (MM_FROUND_TO_ZERO |MM_FROUND_NO_EXC) // truncate, and suppress exceptions MM_FROUND_CUR_DIRECTION // use MXCSR.RC; see MM_SET_ROUNDING_MODE
