gamut.scanline

Scanline conversion public API. This is used both internally and externally, because converting a whole row of pixels at once is a rather common operations.

  • scanlinesConvert

    Declaration

    nothrow @nogc @system bool scanlinesConvert(PixelType srcType, const(ubyte)* src, int srcPitch, PixelType destType, ubyte* dest, int destPitch, int width, int height, PixelType interType, ubyte* interBuf);

    Convert several scanlines, using interBuf an intermediate buffer.

    Parameters

    PixelType srcType

    Type of input pixels.

    const(ubyte)* src

    Input data, point to width x height scanlines.

    int srcPitch

    Bytes between input lines, can be negative.

    PixelType destType

    Type of output pixels.

    ubyte* dest

    Output data, point to width x height scanlines.

    int destPitch

    Bytes between output lines, can be negative.

    int width

    Width of area to convert, in pixels.

    int height

    Height of area to convert, in pixels.

    PixelType interType

    Type given by scanlinesInterType(srcType, destType).

    ubyte* interBuf

    Buffer of pixelTypeSize(interType) * width.

  • scanlineConversionFunction_t

    Declaration

    alias scanlineConversionFunction_t = void function(const(ubyte)* inScan, ubyte* outScan, int width, void* userData) nothrow @nogc @system;

    All scanlines conversion functions follow this signature: inScan pointer to first pixel of input scanline outScan pointer to first pixel of output scanline

    Discussion

    Type information, user data information, must be given by context. Such a function assumes no overlap in memory between input and output scanlines.
